## üíª Project

Local Maps is a simple application to register a commercial establishment according to the local address through a map.

## üîñ Run the project

Clone the project and access its folder.

```bash
$ git clone https://github.com/GabrielDoddy94/localmaps-app
$ cd localmaps-app
```

To start it, follow the steps below:

```bash

# Install the dependencies on backend
$ cd backend
$ yarn

# Start the project on backend
$ yarn dev

# Install the dependencies on frontend
$ cd frontend
$ npm i

# Start the project on backend
$ npm start
```
